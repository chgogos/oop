Μεταγλώττιση του κώδικα για τη main
> g++ linked_list.cpp main.cpp -o main -std=c++11 -Wall -pedantic

Εκτέλεση του κώδικα (σε LINUX)
> ./main

Εκτέλεση του κώδικα (σε windows)
> main.exe

Μεταγλώττιση του κώδικα για τα tests
> g++ linked_list.cpp tests.cpp -o main -std=c++11 -Wall -pedantic

Εκτέλεση του κώδικα (σε LINUX)
> ./tests

Εκτέλεση του κώδικα (σε windows)
> tests.exe
